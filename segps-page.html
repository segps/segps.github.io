SeGPS

SeGPS takes advantage of Semantic Web technologies to establish traceability links between vulnerability repositories and other software repositories using a unified knowledge representation. SE-GPS helps eliminating existing knowledge silos and further allows for information security to become an integrated part of software development processes.
